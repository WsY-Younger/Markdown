### git 入门知识
- git init 创建一个git仓库
- git status 检查一下 看加入没有
- git add 文件名字md
- git commit -m"名字” 提交文件 可以追踪
- git log 贴上标签 随时记录，有生产日期，生产商，改动情况，就是没有保质期哦
- mkdir wsy 创建一个叫wsy的文件夹 且在桌面上
- cd wsy就进去了该文件夹
- ls -a是列出所有内容的意思
- ./和../是两个快捷方式
- cd .进入文件  cd ..进入桌面
- 后缀名要改为md形式，用Atom打开
- 遇到了些问题 有空问一问
- cd 到桌面上 再输cd wsy 竟然说我没有
- 直接输cd wsy也不可。。
- 提交直接在Markdown下进行
